# AstrBot 撤回上一条触发机器人的消息

## 简介

【仅aiocqhttp】撤回上一条触发机器人的消息

## 配置

本插件无需任何配置。

## 注意事项

*   本插件需要 AstrBot 机器人环境。
*   请确保 AstrBot 机器人已正确安装并配置。
*   需要机器人是管理员或群主（有撤回权限）
*   可以在main.py的第55行修改隔多久撤回（建议不超过2分钟）
